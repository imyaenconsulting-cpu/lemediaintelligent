Organiser vos fichiers
Créer la structure de votre projet sur votre ordinateur.
⏱ 10 minutes
📁
Créer un dossier sur votre bureau Nommez-le lemediaintelligent. Mettez-y tous les fichiers HTML reçus (index.html, media.html, skylens.html, guide.html).
📄
Vérifier que vous avez ces 4 fichiers index.html (page d'accueil), media.html (le média), skylens.html (globe 3D), guide.html (ce guide).
✓Vos fichiers sont prêts. Passez à l'étape 2.
2
Créer votre dépôt GitHub
GitHub stocke votre code et permet le déploiement automatique.
⏱ 15 minutes
1️⃣
Aller sur github.com Créez un compte avec votre email Google. Confirmez votre email.
2️⃣
Cliquez sur "New repository" (bouton vert) Nommez-le lemediaintelligent. Choisissez "Public". Cochez "Add a README file". Cliquez "Create repository".
3️⃣
Uploader vos fichiers Dans votre dépôt, cliquez "Add file" → "Upload files". Glissez tous vos fichiers HTML. Cliquez "Commit changes".
💡Une fois vos fichiers uploadés, vous devriez les voir listés dans votre dépôt GitHub. C'est votre code en ligne !
3
Déployer sur Vercel
Vercel met votre site en ligne en 1 clic. Gratuit et ultra-rapide.
⏱ 10 minutes
1️⃣
Aller sur vercel.com Cliquez "Sign Up". Choisissez "Continue with GitHub". Autorisez l'accès.
2️⃣
Importer votre projet Cliquez "New Project". Trouvez "lemediaintelligent" dans la liste. Cliquez "Import".
3️⃣
Configurer et déployer Ne changez rien. Cliquez directement "Deploy". Attendez 1-2 minutes.
4️⃣
Votre site est en ligne ! Vercel vous donne une URL type lemediaintelligent.vercel.app. Cliquez dessus pour voir votre site.
🚀Félicitations ! Votre site est accessible dans le monde entier. Désormais, chaque modification sur GitHub se déploie automatiquement.
4
Connecter votre nom de domaine
Remplacer l'URL Vercel par votre propre domaine professionnel.
⏱ 30 minutes (+ 24h propagation DNS)
1️⃣
Acheter votre domaine sur namecheap.com Cherchez "lemediaintelligent.com". Si disponible (~12€/an), achetez-le. Sinon essayez .fr, .media, ou .info.
2️⃣
Dans Vercel : Settings → Domains Cliquez "Add Domain". Tapez votre domaine. Vercel vous donne des valeurs DNS à copier.
3️⃣
Dans Namecheap : Advanced DNS Ajoutez les enregistrements DNS que Vercel vous a donnés. Attendez 24h maximum.
4️⃣
Créer les sous-domaines Dans Vercel, ajoutez aussi : media.votre-domaine.com et skylens.votre-domaine.com
💡Vercel gère le HTTPS automatiquement. Votre site sera sécurisé (cadenas vert) sans rien faire de plus.
5
Configurer Supabase (base de données)
Supabase stocke vos utilisateurs, articles et données. Gratuit jusqu'à 50 000 utilisateurs.
⏱ 20 minutes
1️⃣
Créer un compte sur supabase.com "Start your project". Connectez avec GitHub. Créez un "New Project". Notez votre mot de passe de base de données.
2️⃣
Activer l'authentification Google Dans votre projet Supabase → Authentication → Providers → Google. Activez-le. Copiez les URLs de redirection.
3️⃣
Récupérer vos clés API Supabase Settings → API. Copiez l'URL du projet et la clé "anon public". Vous en aurez besoin à l'étape 6.
4️⃣
Configurer votre email admin Dans Supabase → SQL Editor, collez et exécutez cette requête pour créer les tables :
-- Créer les tables principales CREATE TABLE users ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), email TEXT UNIQUE, plan TEXT DEFAULT 'free', created_at TIMESTAMP DEFAULT NOW() ); CREATE TABLE articles ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), title TEXT, content TEXT, category TEXT, fact_score INTEGER, ai_generated BOOLEAN DEFAULT false, created_at TIMESTAMP DEFAULT NOW() ); CREATE TABLE fake_news_reports ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), title TEXT, severity TEXT, reach_count INTEGER, created_at TIMESTAMP DEFAULT NOW() ); CREATE TABLE user_preferences ( user_id UUID REFERENCES users(id), topics TEXT[], format TEXT DEFAULT 'text', language TEXT DEFAULT 'fr' );
6
Configurer les clés API
Connecter Claude AI, les données de trafic et les emails.
⏱ 25 minutes
🤖
Clé API Claude (Anthropic) Allez sur console.anthropic.com. Créez un compte. "API Keys" → "Create Key". Copiez la clé (commence par sk-ant-...).
✈️
OpenSky Network (données avions) — GRATUIT Créez un compte sur opensky-network.org. Allez dans "My OpenSky" → "API Credentials". Notez votre username et mot de passe.
📧
Resend (emails automatiques) — GRATUIT Créez un compte sur resend.com. "API Keys" → "Create API Key". Copiez la clé.
🔐
Ajouter les clés dans Vercel (IMPORTANT : jamais dans le code !) Dans Vercel → votre projet → Settings → Environment Variables. Ajoutez une par une :
ANTHROPIC_API_KEY = sk-ant-votre-clé-ici SUPABASE_URL = https://votre-projet.supabase.co SUPABASE_ANON_KEY = votre-clé-anon-supabase OPENSKY_USERNAME = votre-username-opensky OPENSKY_PASSWORD = votre-mot-de-passe-opensky RESEND_API_KEY = re_votre-clé-resend ADMIN_EMAIL = votre-email@gmail.com NEXT_PUBLIC_STRIPE_KEY = pk_live_votre-clé-stripe
⚠️Ne partagez JAMAIS ces clés avec personne. Ne les mettez jamais dans un fichier HTML. Seul Vercel les stocke en sécurité.
7
Configurer les paiements Stripe
Recevoir les paiements Premium et Pro automatiquement.
⏱ 30 minutes
1️⃣
Créer un compte Stripe sur stripe.com Remplissez toutes les informations de votre entreprise (ou en tant que particulier). Vérifiez votre identité (obligatoire pour recevoir des paiements).
2️⃣
Créer vos produits dans Stripe Products → Add product. Créez : "Premium" (19,99€/mois), "Pro" (100€/mois), et chaque module à la carte.
3️⃣
Récupérer vos clés API Stripe Developers → API Keys. Copiez la "Publishable key" (pk_live_...) et la "Secret key" (sk_live_...). Ajoutez-les dans Vercel.
4️⃣
Configurer les Webhooks Developers → Webhooks → Add endpoint. URL : https://votre-domaine.com/api/stripe-webhook. Sélectionnez les événements : checkout.session.completed, customer.subscription.updated.
💡Stripe vous vire automatiquement l'argent sur votre compte bancaire tous les 7 jours. Vous pouvez voir vos revenus en temps réel dans le dashboard Stripe.

🆘 Support — Résolution de problèmes

Si quelque chose ne fonctionne pas, cherchez votre problème ici.

⚠️ Le site ne se charge pas
Vous voyez une erreur 404 ou page blanche.
✓ Vérifiez que vos fichiers sont bien dans GitHub. Allez dans Vercel → votre projet → Deployments et vérifiez que le dernier déploiement est "Ready".
⚠️ Le domaine ne fonctionne pas
Votre domaine personnalisé affiche une erreur.
✓ La propagation DNS peut prendre jusqu'à 48h. Attendez et réessayez. Vérifiez dans Vercel → Domains que le statut est "Valid".
⚠️ L'IA ne répond pas
L'assistant IA ne génère pas de réponses.
✓ Vérifiez votre clé API Anthropic dans Vercel → Environment Variables. Assurez-vous d'avoir du crédit sur console.anthropic.com.
⚠️ La connexion Google échoue
Le bouton "Continuer avec Google" ne fonctionne pas.
✓ Dans Supabase → Authentication → Google, vérifiez que l'URL de redirection correspond exactement à votre domaine Vercel.
⚠️ Le paiement Stripe échoue
Les utilisateurs ne peuvent pas payer.
✓ Vérifiez que vous êtes en mode "Live" dans Stripe (pas "Test"). Assurez-vous que votre identité est vérifiée dans Stripe.
⚠️ SkyLens ne montre pas les avions
Le globe 3D est vide ou ne charge pas les données.
✓ Vérifiez vos identifiants OpenSky dans Vercel. L'API OpenSky a une limite de requêtes — attendez 10 secondes entre les actualisations.
⚠️ Les emails n'arrivent pas
Les newsletters et notifications ne sont pas envoyées.
✓ Dans Resend, vérifiez que votre domaine d'envoi est validé. Ajoutez les enregistrements DNS SPF et DKIM dans Namecheap.
⚠️ Site lent ou saturé
Le site rame avec beaucoup d'utilisateurs.
✓ Activez Cloudflare (gratuit) devant votre domaine. Il agit comme un bouclier anti-DDoS et met en cache votre site dans le monde entier.
⚠️ Je ne peux pas accéder à l'admin
Le dashboard admin n'est pas accessible.
✓ Vérifiez que votre email dans ADMIN_EMAIL (Vercel) correspond exactement à votre email Google de connexion. Sensible à la casse.
⚠️ Erreur 500 serveur
Erreur interne du serveur.
✓ Dans Vercel → votre projet → Logs, regardez les erreurs en rouge. Copiez le message d'erreur et recherchez-le sur Google ou contactez le support Vercel.
