🆘 Résolution de problèmes

Si quelque chose ne fonctionne pas, trouvez votre problème ici.

⚠️ Site ne charge pas (404)
Vous voyez une page d'erreur Vercel.
✓ Vérifiez GitHub que vos fichiers sont présents. Dans Vercel → Deployments → vérifiez que le statut est "Ready" en vert.
⚠️ Domaine ne fonctionne pas
Votre domaine affiche une erreur ou ne redirige pas.
✓ Attendez 24-48h. Vérifiez dans Vercel → Domains que le statut est "Valid". Vérifiez les enregistrements DNS dans Namecheap.
⚠️ L'IA ne répond pas
L'assistant reste silencieux ou affiche une erreur.
✓ Vérifiez ANTHROPIC_API_KEY dans Vercel. Vérifiez que vous avez du crédit sur console.anthropic.com → Billing.
⚠️ Connexion Google échoue
Le bouton Google ne fonctionne pas ou redirige mal.
✓ Dans Supabase → Authentication → Google, vérifiez que l'URL de redirection correspond exactement à votre domaine Vercel.
⚠️ Globe 3D vide ou noir
SkyLens ne charge pas le globe ou il est noir.
✓ Vérifiez que Three.js se charge (connexion internet requise). Essayez dans Chrome. Videz le cache (Ctrl+Shift+R).
⚠️ Paiement Stripe échoue
Les utilisateurs ne peuvent pas payer.
✓ Vérifiez que vous êtes en mode "Live" dans Stripe (pas "Test"). Assurez-vous que votre identité est vérifiée dans Stripe → Settings.
⚠️ Emails non reçus
Les newsletters n'arrivent pas.
✓ Dans Resend, vérifiez que votre domaine d'envoi est validé (DNS SPF + DKIM). Vérifiez les spams.
⚠️ Erreur 500 serveur
Erreur interne mystérieuse.
✓ Dans Vercel → votre projet → Logs. Lisez le message d'erreur en rouge. Copiez-le et soumettez-le à Claude pour diagnostic.
